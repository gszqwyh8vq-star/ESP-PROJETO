# FreefireESP_Satooios

Este proyecto es un **tweak de iOS** diseñado para el juego **Garena Free Fire**. Su función principal es proporcionar características de **ESP (Extra Sensory Perception)**, como líneas y cajas alrededor de los jugadores, para mejorar la experiencia de juego.

El tweak se inyecta en la aplicación de Free Fire en dispositivos iOS con jailbreak y ofrece un menú flotante para activar y desactivar las funciones.

## Características Principales

*   **Menú Flotante:** Un menú interactivo que se activa con un gesto de **dos toques con tres dedos** y se cierra con **dos toques con dos dedos**.
*   **Activación de Cheats:** Un interruptor principal para habilitar o deshabilitar todas las funciones del tweak.
*   **ESP de Líneas:** Muestra líneas que conectan al jugador con los enemigos.
*   **ESP de Cajas:** Dibuja cajas alrededor de los modelos de los jugadores enemigos.
*   **Seguridad:** Incluye funciones de ofuscación de código (`mahoa.h`, `Obfuscate.h`) y encriptación AES para proteger ciertas cadenas de texto y dificultar la ingeniería inversa.

## Requisitos de Compilación

Para compilar este proyecto, necesitarás:

1.  **Theos:** El entorno de desarrollo para tweaks de iOS.
2.  Un dispositivo iOS con **Jailbreak** para la instalación y prueba.
3.  Las herramientas de desarrollo de Apple (Xcode).

## Instalación

1.  Asegúrate de tener **Theos** configurado correctamente.
2.  Clona o descarga este repositorio.
3.  Abre una terminal en el directorio del proyecto.
4.  Ejecuta `make package install` para compilar e instalar el tweak en tu dispositivo.
